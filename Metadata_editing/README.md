## Metadata Editing

### ID3 tags
[**mid3v2**](https://mutagen.readthedocs.io/en/latest/man/mid3v2.html)
```sh
pip install mutagen

# list all tags
mid3v2 -l /path/file

# write tag
mid3v2 
```
--artist=ARTIST (TPE1)
--album=ALBUM (TALB)
--song=TITLE (TIT2)
--genre=GENRE (TCON)
--year=YEAR (TDRC)
--track=NUM (TRCK)
